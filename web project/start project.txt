Создание самого проекта
1 открыть папку, в которой будет храниться проект
2 запустить эту папку черед cmd
3 написать в cmd> django-admin startproject (название нового проекта)

создание env(сохранинение необходимой версии продукта дял работы)




Создание приложения 
1 открыть папку, в которой хранится manage.py
2 cmd
3 сmd>python manage.py startapp (название приложения)
4 proj-settings-installed_apps (добавит свое приложение по имени('app'))

для того чтоб просмотреть сайт
1 открыть папку, в которой хранится manage.py
2 сmd>python manage.py runserver
3 копируем http и вставляем в браузер
